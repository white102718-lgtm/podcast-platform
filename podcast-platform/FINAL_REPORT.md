## 📝 使用流程

### 1. 首次使用
1. 打开应用
2. 设置 API Keys（OpenAI + Anthropic）
3. 密钥保存在浏览器本地

### 2. 创建项目
1. 点击"+ 新建项目"
2. 输入项目名称
3. 开始编辑

### 3. 上传音频
1. 选择项目
2. 拖拽或选择音频文件
3. 等待 AI 转录

### 4. 编辑音频
- 点击词语跳转音频位置
- 使用快捷操作（去口头禅、去静音）
- 使用音频增强（EQ、音频平衡）
- 查看编辑统计

### 5. 导出音频
1. 点击"导出 MP3"
2. 等待处理完成
3. 下载文件
