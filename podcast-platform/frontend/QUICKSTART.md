## 📝 使用流程

### 1. 首次使用
1. 打开应用，会弹出 API Keys 设置
2. 输入 OpenAI API Key（必填）
3. 输入 Anthropic API Key（可选）
4. 点击"保存并开始"

### 2. 创建项目
1. 点击左侧边栏"+ 新建项目"
2. 输入项目名称
3. 点击"创建"

### 3. 上传音频
1. 选择项目
2. 拖拽音频文件或点击选择
3. 等待上传和转录完成

### 4. 编辑音频
1. 在词级标注视图中查看转录内容
2. 点击词语跳转到对应音频位置
3. 使用右侧快捷操作：
   - 去口头禅
   - 去静音段
   - 撤销上一步

### 5. 导出音频
1. 点击右侧"导出 MP3"
2. 等待处理完成
3. 点击"下载 MP3"

## 🐛 故障排除

### 开发服务器无法启动
```bash
# 清理缓存
rm -rf node_modules
npm install
npm run dev
```

### 构建失败
```bash
# 检查 TypeScript 错误
npm run build
```

### API 连接失败
1. 检查后端是否运行（http://localhost:8000）
2. 检查 `.env` 文件中的 `VITE_API_BASE_URL`
3. 检查 API Keys 是否正确设置
